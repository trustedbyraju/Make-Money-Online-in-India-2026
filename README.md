# Make Money Online in India 2026 — Parasite SEO Site

Single-page HTML5 + CSS + JS site optimized to rank for **"Make Money Online in India 2026"** on GitHub Pages.

## 🚀 Deploy in 3 minutes

1. Create a new GitHub repo named: `make-money-online-india-2026` (the repo name becomes part of the URL — keep the keyword!).
2. Upload `index.html`, `robots.txt`, `sitemap.xml`.
3. Go to **Settings → Pages → Source: main / root** → Save.
4. Your site will be live at: `https://YOURUSERNAME.github.io/make-money-online-india-2026/`
5. Find-replace `yourusername` in `index.html`, `robots.txt`, `sitemap.xml` with your actual GitHub username.
